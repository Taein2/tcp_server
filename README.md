# tcp_server
socket connection

UNIX환경에서 사용 가능한 tcp_ser.c 파일입니다.  
inet.h 헤더파일
------------------------------------------------------------------  
Concurrent 병행 서버 
- fork()함수를 이용(다중 클라이언트 연결)   
- while문을 통한 client와의 계속된 응답  
- 클라이언트에서 온 덧셈을 처리  
 ex) 3+4  
     5+8  
     10+232
